Impact of weather events on health and economic in the USA.

Storms and other severe weather events can cause both public health and economic problems for communities and municipalities. Many severe events can result in fatalities, injuries, and property damage, and preventing such outcomes to the extent possible is a key concern.

In this project it is explored different characteristics and estimates of storm on storm database.

Synopsis

Tornadoes are the most dangerous weather event to the population health. Next it is excessive heat. Flash floods and thunderstorm winds caused a lot of money due to damage.

Most economic difficult (property damage, crop damage) event types

To analyze the impact of weather events on the economy, available property damage and crop damage reportings/estimates were used. Transformation h -> hundred, k -> thousand, m -> million, b -> billion

exp_transform <- function(e) {
    if (e %in% c('h', 'H'))
        return(2)
    else if (e %in% c('k', 'K'))
        return(3)
    else if (e %in% c('m', 'M'))
        return(6)
    else if (e %in% c('b', 'B'))
        return(9)
    else if (!is.na(as.numeric(e))) # if a digit
        return(as.numeric(e))
    else if (e %in% c('', '-', '?', '+'))
        return(0)
    else {
        stop("Invalid exponent value.")
    }
}

Damage is calculated

prop_dmg_exp <- sapply(storm$PROPDMGEXP, FUN=exp_transform)
storm$prop_dmg <- storm$PROPDMG * (10 ** prop_dmg_exp)
crop_dmg_exp <- sapply(storm$CROPDMGEXP, FUN=exp_transform)
storm$crop_dmg <- storm$CROPDMG * (10 ** crop_dmg_exp)

Compute the economic loss by event type and filter out events that caused no economic loss

library(plyr)
econ_loss <- ddply(storm, .(EVTYPE), summarize,
                   prop_dmg = sum(prop_dmg),
                   crop_dmg = sum(crop_dmg))

econ_loss <- econ_loss[(econ_loss$prop_dmg > 0 | econ_loss$crop_dmg > 0), ]
prop_dmg_events <- head(econ_loss[order(econ_loss$prop_dmg, decreasing = T), ], 10)
crop_dmg_events <- head(econ_loss[order(econ_loss$crop_dmg, decreasing = T), ], 10)

Results

Health impact of weather events

The plot shows most dangerous weather event types. (injuries, fatalities)

library(ggplot2)
library(gridExtra)
## Loading required package: grid
p1 <- ggplot(data=fatal_events,
             aes(x=reorder(EVTYPE, fatalities), y=fatalities, fill=fatalities)) +
    geom_bar(stat="identity") +
    coord_flip() +
    ylab("Number of fatalities") +
    xlab("Weather event type") +
    theme(legend.position="none")

p2 <- ggplot(data=injury_events,
             aes(x=reorder(EVTYPE, injuries), y=injuries, fill=injuries)) +
    geom_bar(stat="identity") +
    coord_flip() + 
    ylab("Number of injuries") +
    xlab("Weather event type") +
    theme(legend.position="none")

grid.arrange(p1, p2, main="The most dangerous weather events in the USA")

The most dangerous weather events in USA that cause death and injuries are tornadoes. They caused more than 5000 deaths and more than 10000 injuries for 60 years. The second and third place are excessive heat and flash floods….
